# SONYACHNA-VENUE


## DESCRIPTION
This is a backtesting engine responsible for executing the backtesting process. It takes the processed data points and applies the defined trading strategy to simulate trades and generate results. Simulation is very life-like, it simulates things like slippage, commission, portfolio management, broker simulation. The engine evaluates basic performance metrics such as total return, maximum drawdown, before sending the results to a node that will compute more advanced metrics and generate visualizations.
